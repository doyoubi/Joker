## 《Joker》 手游

《Joker》是一个结合了节奏元素的平台游戏，玩家需要跟着节拍指示控制人物移动、躲避机关、击倒敌人。

[游戏视频](http://v.youku.com/v_show/id_XOTA3ODU2MjI0.html)

开发人员：

- 美术：屠秋晓
